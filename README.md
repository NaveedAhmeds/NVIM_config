# Neovim: "Hold My Coffee" Edition
### (Go + Rust + Extreme Laziness)

A specialized Neovim setup powered by **LazyVim**, meticulously tuned for Gophers and Rustaceans. It’s fast, solid (literally no transparency), and designed for people who prefer writing code over fighting their editor.


##  The "Secret Sauce"

* **Languages:** Native-level support for **Go** (real tabs, because we aren't monsters) and **Rust**.
* **Vibes:** Zero transparency. Solid UI for solid code.
* **Speed:** `Hop.nvim` for teleporting around files and `auto-save` because I have commitment issues.
* **Utilities:** Floating terminals via `Snacks`, custom statuslines, and `todo-comments` to track my inevitable technical debt.
